# VulkanRenderer
Vulkan Renderer

### Installation
1. Clone the repo
```sh
git clone --recursive https://github.com/sebimih13/VulkanRenderer.git
```

2. Generate Projects
```sh
GenerateProjects.bat
```

3. Build and Run VulkanRenderer.sln
```sh
VulkanRenderer.sln
```
